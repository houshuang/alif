Educational HTML reference pages live in `research/` and are indexed in `research/research-hub.html` under the "Arabic Reference Pages" category.

**Pages:**
- `research/quran-function-words.html` — 70+ Quranic function words with register tags (All Arabic / Classical & Formal MSA / Quranic), verse examples, grammar notes, clitic decomposition diagrams
- `research/arabic-ligatures-reference.html` — Arabic letter connections, ligature rules, initial/medial/final forms
- `research/quranic-marks-reference.html` — Quranic diacritical marks, tajweed notation, waqf signs, tashkeel
- `research/arabic-font-comparison.html` — Side-by-side comparison of Arabic fonts (Scheherazade, Amiri, Noto)

**How to apply:** When creating new Arabic educational content, add it to this collection and register it in `research-hub.html` under the "Arabic Reference Pages" category. Use gold/amber accent color and dark theme matching the existing pages.
