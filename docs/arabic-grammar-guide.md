# Arabic Grammar Guide for Reading Comprehension

Grammar concepts organized by learning tier, focused on what a reader needs to recognize Arabic text. Each concept is tagged as either **form-changing** (alters word shape, blocks recognition) or **structural** (adds/rearranges words).

## Key Recognition Challenges

### Form-Changing Concepts (block word recognition)
These alter the visual shape of words, making it hard to connect surface forms to known lemmas:
1. **Proclitic prepositions** — بالكتاب looks like one word but is بـ + ال + كتاب
2. **Attached pronouns** — كتابه looks different from كتاب
3. **Broken plurals** — كُتُب bears little visual resemblance to كِتَاب
4. **Verb conjugation** — يَكْتُبُونَ has 3 morphemes around the root
5. **Active/passive participles** — كاتِب and مَكْتُوب are derived from كَتَبَ

### Structural Concepts (change sentence structure)
These don't change word shapes but affect how sentences are parsed:
1. **Nominal vs verbal sentences** — Arabic can start with either subject or verb
2. **Kāna/Inna and sisters** — frequently used, change expected case
3. **Idafa** — possessive construct without any visible marker
4. **Relative clauses** — الذي etc. link clauses differently from English
